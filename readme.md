# conecto to reds

```
docker exec -it redis redis-cli
```

### Set data into Redis

```sh
SET key1 "value1"
SET key2 "value2"
SET key3 "value3"
SET key4 "value4"
SET key5 "value5"
```

### Get data from Redis

```sh
GET key1
GET key2
GET key3
GET key4
GET key5
```

### Delete specific keys from Redis

```sh
DEL key1
DEL key2
```

### Delete all keys from Redis

```sh
FLUSHALL
```
